# Garment Category & Stitching Style Recognition

A Hybrid Transfer Learning based Deep Learning system for automated garment category and stitching style recognition.

---

## Project Overview

This project focuses on automating the classification of garments and identifying stitching styles using Deep Learning and Transfer Learning techniques.  
It predicts both **garment category** and **stitching condition** from input images, helping improve quality control in the textile and garment industry.

---

## Features

- Automatic garment category classification  
- Stitching style / defect recognition  
- Hybrid Transfer Learning approach  
- Multi-output deep learning model  
- Faster training using pre-trained CNN models  
- Scalable for real-world industrial use
